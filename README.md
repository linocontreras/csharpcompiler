# C# Interactive Compiler

## Dependencies
For installing dependencies run
```
pip install -r requirements.txt
```

## Usage

```
./csc.py inputfile.cs
```

For printing the AST add the -ast option

```
./csc.py -ast inputfile.cs
```

## Tests

For runnning the test suite:
```
./test.sh
```
